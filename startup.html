Startup fit

I built a quant fund with 3 people that does what usually takes 50. I know what building from nothing looks like.

Event Horizon Partners

Three people. Seven years. We built an institutional-grade quantitative trading platform from scratch. ML model training and orchestration, live execution, investor operations automation, risk management. Thousands of models running every paradigm — supervised, unsupervised, reinforcement learning, clustering. Sub-millisecond inference. Sharpe ratio above 6.

I built the whole technology stack. Python, C#, TypeScript, BigQuery, DuckDB (100x faster than the MongoDB setup it replaced), Docker, Terraform, Ray Tune for hyperparameter optimization. 33 production repositories. Zero downtime for 7 years.

Building treasury from scratch

I built a production treasury and payment operations platform in 3 months. Double-entry immutable ledger, multi-rail payments (ACH, Wire, RTP), JPMorgan Chase Wallet API integration with V2V transfers and virtual accounts, Stripe checkout, Plaid bank verification, reconciliation engine, approval workflows, waterfall revenue distribution, webhook delivery, QuickBooks GL sync.

40 database tables. 109 API endpoints. 14 service modules. Provider abstraction so you can swap JPMorgan for US Bank or Stripe for Adyen without touching business logic. Two-phase payment pattern that keeps the ledger consistent even when provider calls fail. Transaction-level reconciliation with auto-matching. All 23 ACH return codes handled automatically.

FastAPI, PostgreSQL, SQLAlchemy, Terraform, GCP Cloud Run. The kind of thing Modern Treasury sells as a product. I built it as infrastructure.
